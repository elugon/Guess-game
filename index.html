Guess the Animals

Instructions

  1. Click on each card and be able to see it.
  2. If the user clicks on 3 cards of the same type in a row this cards will get pop of the table.
  3. If the user pop off all the cards from the table before the counter reach 0 seg the user will win the game, otherwise, it will loose.
  4. If only one click has been done that first card remains visible.
  5. If the second card dont belong to the type-card of the first card clicked both cards return to a not visible state
